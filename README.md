## 📦 Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/nextpointer/n0t3s.git

# 2️⃣ Install dependencies with Bun
bun install

# 3️⃣ Set up environment variables
cp .env.example .env.local

# 4️⃣ Run the development server
bun dev
```
