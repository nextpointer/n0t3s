## 📦 Installation

```bash
# 1️⃣ Clone the repository
git clone https://github.com/nextpointer/next-notes.git

# 2️⃣ Install dependencies with Bun
bun install

# 3️⃣ Run the development server
bun dev
```

- ➕ Click Add Note
- 🖊 Write your note & add tags
- 💾 Notes are saved automatically
